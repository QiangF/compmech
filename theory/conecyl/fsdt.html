FSDT - First-order Shear Deformation Theory¶

Description¶

For the FSDT the displacement field components are:

\[u, v, w, \phi_x, \phi_t\]

The displacement field is approximated similarly to the CLPT, but here \(\phi_x\) and \(\phi_\theta\) are independent variables, relaxing the approximation.

For the ConeCyl implementations, the approximation functions are separated into three components:

\[\begin{split}u = u_0 + u_1 + u_2\\\end{split}\]\[\vdots\]\[{{\phi}_\theta} = {{\phi}_\theta}_0 + {{\phi}_\theta}_1 + {{\phi}_\theta}_2\]

where \(u_0\) contains the approximation functions corresponding to the prescribed degrees of freedom, \(u_1\) contains the functions independent of \(\theta\) and \(u_2\) the functions that depend on both \(x\) and \(\theta\).

The aim is to have a model capable to simulate non-rigid supports, and where the displacement components \(u, \phi_x\) can habe a non-costant value along the edges.

It is of special importance to allow \(\phi_x\) to be between zero (clamped) and another value (up to simply supported), by using elastic stiffnesses for the corresponding degrees of freedom.

Models¶

The recommended model is the fsdt_donnell5 (or the counterpart fsdt_sanders5) which has the largest simulation capabilities among the models and showed to be stable for linear buckling analysis too. Moreover, this was achieved using an approximation function which requires less Ritz constants to predict the same behavior. The other models are kept here for future reference.

Below it follows are listed below (same name as the linear_kinematics parameter of the ConeCyl object):

  • fsdt_donnell
  • fsdt_donnell2
  • fsdt_donnell3
  • fsdt_donnell4
  • fsdt_donnell5
  • fsdt_donnell6
  • fsdt_sanders5

fsdt_donnell¶

The approximation functions are:

\[\begin{split}u = u_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}_a^{u} \sin{{b_x}_1} +{c_{i_1}}_b^{u} \cos{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{u} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{u} \sin{{b_x}_2} \cos{j_2 \theta} +{c_{i_2 j_2}}_c^{u} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_d^{u} \cos{{b_x}_2} \cos{j_2 \theta} \right) \\ v = v_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{v}\sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{v} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{v} \sin{{b_x}_2} \cos{j_2 \theta} \right) \\ w = w_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{w}\sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{w} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{w} \sin{{b_x}_2} \cos{j_2 \theta} \right) \\ \phi_x = {\phi_x}_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{\phi_x}_a\sin{{b_x}_1} +{c_{i_1}}^{\phi_x}_b\cos{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{\phi_x} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{\phi_x} \sin{{b_x}_2} \cos{j_2 \theta} +{c_{i_2 j_2}}_c^{\phi_x} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_d^{\phi_x} \cos{{b_x}_2} \cos{j_2 \theta} \right) \\ \phi_{\theta} = {\phi_{\theta}}_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{\phi_{\theta}}_a\sin{{b_x}_1} +{c_{i_1}}^{\phi_{\theta}}_b\cos{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{\phi_{\theta}} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{\phi_{\theta}} \sin{{b_x}_2} \cos{j_2 \theta} +{c_{i_2 j_2}}_c^{\phi_{\theta}} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_d^{\phi_{\theta}} \cos{{b_x}_2} \cos{j_2 \theta} \right)\end{split}\]

with:

\[\begin{split}{b_x}_1 = i_1 \pi \frac x L \\ {b_x}_2 = i_2 \pi \frac x L\end{split}\]

Observations:

\(\checkmark\) linear static

\(\checkmark\) succcessfuly simulated the linear static loads

\(\checkmark\) the flexible boundary conditions were successfully simulated

\(\times\) rigid body translations in \(u\)

\(\times\) did not work for linear buckling

\(\times\) did not work for non-linear analysis

fsdt_donnell2¶

With the aim to remove the rigid body translation found in fsdt_donnell, only the function for \(u\) has been changed:

\[u = u_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{u}\sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{u} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{u} \sin{{b_x}_2} \cos{j_2 \theta} +{c_{i_2 j_2}}_c^{u} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_d^{u} \cos{{b_x}_2} \cos{j_2 \theta} \right)\]

Observations:

\(\checkmark\) rigid body translation removed

\(\times\) did not work for linear buckling

\(\rightarrow\) non-linear analysis not implemented

fsdt_donnell3¶

Tried to remove some terms of fsdt_donnell2 in order to make the linear buckling analysis work. Only the \(cos\) function of \({\phi_x}_1\) has been removed:

\[\phi_x = {\phi_x}_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{\phi_x} \sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{\phi_x} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{\phi_x} \sin{{b_x}_2} \cos{j_2 \theta} +{c_{i_2 j_2}}_c^{\phi_x} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_d^{\phi_x} \cos{{b_x}_2} \cos{j_2 \theta} \right)\]

Observations:

\(\times\) did not work for linear buckling

\(\rightarrow\) non-linear analysis not implemented

fsdt_donnell4¶

Tried to remove some terms of fsdt_donnell3. The \(cos\) functions of \(u_2\) was removed, the \(cos\) function of \({\phi_x}_1\) was put back:

\[\begin{split}u = u_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{u}\sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{u} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{u} \sin{{b_x}_2} \cos{j_2 \theta} \right) \\ \phi_x = {\phi_x}_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}_a^{\phi_x} \sin{{b_x}_1} +{c_{i_1}}_b^{\phi_x} \sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{\phi_x} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{\phi_x} \sin{{b_x}_2} \cos{j_2 \theta} +{c_{i_2 j_2}}_c^{\phi_x} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_d^{\phi_x} \cos{{b_x}_2} \cos{j_2 \theta} \right)\end{split}\]

Observations:

\(\times\) removing the \(cos\) for \(u_2\) removed the capability to simulate non-rigid boundary conditions in \(u\)

\(\times\) did not work for linear buckling

\(\rightarrow\) non-linear analysis not implemented

fsdt_donnell5¶

From the four previous attempts, the fsdt_donnell2 and fsdt_donnell3 give the same results and the fsdt_donnell3 is preferred because it has less degrees of freedom (the \(cos\) of \({\phi_x}_1\)). By then it was learned how the approximations for non-rigid boundary conditions should be and the current attempt add more flexibility in \(v,w,\phi_\theta\) using the previous know-how. The resulting approximation functions are:

\[\begin{split}u = u_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{u} \sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{u} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{u} \cos{{b_x}_2} \cos{j_2 \theta} \right) \\ v = v_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{v} \sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{v} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{v} \cos{{b_x}_2} \cos{j_2 \theta} \right) \\ w = w_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{w} \sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{w} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{w} \cos{{b_x}_2} \cos{j_2 \theta} \right) \\ \phi_x = {\phi_x}_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{\phi_x} \cos{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{\phi_x} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{\phi_x} \cos{{b_x}_2} \cos{j_2 \theta} \right) \\ {\phi}_\theta = {\phi_x}_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{{\phi}_\theta} \sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{{\phi}_\theta} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{{\phi}_\theta} \cos{{b_x}_2} \cos{j_2 \theta} \right)\end{split}\]

Observations:

\(\checkmark\) linear static implemented

\(\checkmark\) linear buckling implemented

\(\rightarrow\) non-linear analysis not implemented

fsdt_donnell6¶

With an attempt to make the linear buckling analysis to work for FSDT, this uses very simple approximation functions, design to produce results for a simply supported cone or cylinder, analogous to the CLPT implementation:

\[\begin{split}u = u_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{u} \sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{u} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{u} \sin{{b_x}_2} \cos{j_2 \theta} \right) \\ v = v_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{v} \sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{v} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{v} \sin{{b_x}_2} \cos{j_2 \theta} \right) \\ w = w_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{w} \sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{w} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{w} \sin{{b_x}_2} \cos{j_2 \theta} \right) \\ \phi_x = {\phi_x}_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{\phi_x} \cos{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{\phi_x} \cos{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{\phi_x} \cos{{b_x}_2} \cos{j_2 \theta} \right) \\ {\phi}_\theta = {\phi_x}_0 + \sum_{i_1=i_0}^{m_1} {c_{i_1}}^{{\phi}_\theta} \sin{{b_x}_1} + \sum_{i_2=i_0}^{m_2} \sum_{j_2=i_0}^{n_2} \left( {c_{i_2 j_2}}_a^{{\phi}_\theta} \sin{{b_x}_2} \sin{j_2 \theta} +{c_{i_2 j_2}}_b^{{\phi}_\theta} \sin{{b_x}_2} \cos{j_2 \theta} \right)\end{split}\]
